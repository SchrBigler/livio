# LIVIO – Biglers Insektenschutz-Lotse

Pilot-Webapp fuer die Schreinerei Bigler.

## Ziel
Ein Kunde kann sein Projekt erfassen, Bereiche anlegen, Bauteile erfassen und eine erste Kostenschaetzung erhalten.

## Lokal starten
```bash
npm install
npm run dev
```
Dann im Browser oeffnen:

http://localhost:3000

## Deployment
Empfohlen: GitHub + Vercel.

1. Repository auf GitHub erstellen
2. Dateien hochladen oder pushen
3. In Vercel "New Project" waehlen
4. GitHub-Repository verbinden
5. Deploy klicken

Danach gibt es einen Testlink fuer Handy, Tablet und PC.
